# 1-Premier-deploiement

Ce projet a pour but pour de récupérer une architecture de dossier terraform existante pour commencer à prendre en main l'outil 

## Pour commencer, récupérer ce projet sur votre poste
Remplacer le valeur <PAT> par celui que je vous donnerais
```
git clone https://oauth2:<PAT>@gitlab.com/mentorat_terrafom/1-premier-deploiement.git
```

## Composition du projet
Il y a trois branches :
- La première **main** permet d'avoir les fichiers de base pour pratiquer les commandes terraform et comprendre le workflow terraform
- La deuxième **use_variable** permet de comprendre l'utilisation des variables et commencer à voir l'utilisation des modules
- La troisième extia-lab permet le déploiement sur le lab d'extia (serveur proxmox)

Enjoy 🥳
